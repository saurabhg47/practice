1. For checking django version:
    $ python -c "import django; print(django.get_version())"

2. For creating project directory:
    $ django-admin startproject mysite (project_name)

3. For install whatever you configured with in INSTALLED_APPS section of setting.py:
    $ python manage.py migrate (again run after 7th cmd to create table in DB)

4. For run server:
    $ python manage.py runserver (default 127.0.0.1:8000)
    $ python manage.py runserver 8080 (specific port_no)
    $ python manage.py runserver 0.0.0.0:8000 (listen on all public IPs (useful if you want to show off your work on other computers on your network)

5. For creating app directory :
    $ python manage.py startapp polls (app_name)

6. For intalled app which we created above:
    $ python manage.py makemigrations polls (app_name)

7. For sqlmigrate takes migration names and returns their SQL:
    $ python manage.py sqlmigrate polls 0001

8. For checking errors in project:
    $ python manage.py check;

6. For finding record in a column which will start with:
>>> Question.objects.filter(question_test__startswith='What')

>>> Question.objects.get(id=1)
